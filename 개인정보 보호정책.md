---

## 1. 개인정보의 처리 목적

주희봇('https://github.com/kevin1113-github/juhee-bot' 이하 '주희봇')은(는) 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 개인정보 보호법 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

1. **텍스트 음성 변환(TTS) 서비스 제공**
   - Discord 서버 내 텍스트 메시지를 음성으로 변환하여 음성 채널에서 재생
   - 사용자별 맞춤 음성 설정 제공 (목소리, 속도)

2. **서비스 운영 및 관리**
   - 서버별 TTS 채널 설정 관리
   - 사용자별 설정 저장 및 관리
   - 서비스 이용 기록 관리 (로그)

## 2. 처리하는 개인정보의 항목

주희봇은 다음과 같은 개인정보를 처리합니다:

### 가. 필수 수집 항목
1. **Discord 사용자 정보**
   - Discord 사용자 고유 ID
   - Discord 사용자명 (username)
   - Discord 서버 내 표시 이름 (displayName)
   - Discord 서버 내 글로벌 이름 (globalName)

2. **Discord 서버 정보**
   - Discord 서버 고유 ID
   - 서버 이름
   - 서버 멤버 수

3. **메시지 정보**
   - TTS 채널에 전송된 메시지 내용 (최대 200자로 제한)
   - 메시지 전송 시각
   - 메시지 전송 채널 ID

4. **사용자 설정 정보**
   - 선호하는 TTS 음성 (예: SeoHyeonNeural, InJoonNeural 등)
   - TTS 속도 설정 (0-100 범위)

5. **서버 설정 정보**
   - TTS 활성화 채널 ID
   - 음소거 설정 여부

### 나. 자동 수집 항목
1. **서비스 이용 기록**
   - 봇 명령어 실행 기록
   - 음성 채널 입/퇴장 기록
   - 오류 발생 기록
   - 서비스 접속 로그 (타임스탬프, 서버 ID, 사용자 ID)

2. **음성 연결 정보**
   - 음성 채널 ID
   - 음성 연결 상태
   - 음성 채널 참여자 정보

## 3. 개인정보의 처리 및 보유 기간

주희봇은 법령에 따른 개인정보 보유·이용기간 또는 정보주체로부터 개인정보를 수집 시에 동의받은 개인정보 보유·이용기간 내에서 개인정보를 처리·보유합니다.

### 가. 즉시 삭제 항목
- **메시지 내용**: TTS 음성 변환 완료 후 메모리에서 즉시 삭제됨 (저장되지 않음)
- **음성 데이터**: Microsoft Azure TTS API를 통해 생성된 음성 데이터는 재생 후 즉시 삭제됨

### 나. 서비스 이용 기간 동안 보유
- **사용자 ID 및 설정**: 서비스 이용 종료 시 또는 사용자 삭제 요청 시까지 보관
- **서버 정보**: 봇이 서버에서 제거될 때까지 보관 (서버 퇴장 시 자동 삭제)
- **서버-사용자 관계 정보**: 봇이 서버에서 제거되거나 사용자 삭제 요청 시까지 보관

### 다. 로그 파일 보관 기간
- **서비스 이용 로그**: 생성일로부터 **7일간 보관** 후 자동 삭제
- **로그 저장 위치**: 로컬 파일 시스템 (개발 모드: logs-dev/, 프로덕션 모드: logs-prod/)
- **로그 내용**: 
  - 서비스 시작/종료 시각
  - 명령어 실행 기록 (사용자 ID, 서버 ID, 명령어 종류)
  - 오류 발생 기록
  - 음성 채널 입/퇴장 기록
  - TTS 요청 기록 (사용자 ID, 서버 ID, 텍스트 길이, 목소리 설정)

## 7. 개인정보의 파기절차 및 파기방법

주희봇은 개인정보 보유기간의 경과, 처리목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체없이 해당 개인정보를 파기합니다.

### 가. 파기절차
1. 불필요한 개인정보는 개인정보 보호책임자의 책임 하에 내부 방침 절차에 따라 파기합니다
2. 서버에서 봇이 제거될 경우 해당 서버의 모든 데이터가 자동으로 파기됩니다
3. 사용자가 삭제를 요청할 경우 요청 접수 후 **7일 이내**에 파기합니다

### 나. 파기방법
1. **데이터베이스 저장 정보**
   - SQLite 데이터베이스에서 해당 레코드 완전 삭제 (CASCADE 삭제 적용)
   
2. **로그 파일**
   - 7일 경과 후 자동으로 파일 시스템에서 영구 삭제
   
3. **메모리 데이터**
   - 프로세스 종료 시 자동으로 메모리에서 제거

### 다. 자동 파기 시스템
- **봇 서버 퇴장 시**: 해당 서버의 모든 데이터 자동 삭제
- **로그 파일**: 생성일로부터 7일 후 자동 삭제
- **음성 데이터**: 재생 완료 즉시 메모리에서 삭제
